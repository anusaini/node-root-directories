# node-root-directories

Get root directories for machine - node module

## Usage

```javascript

  var rd = require('./')

  > rd()
  [ 'Applications',
    'Developer',
    'Library',
    'Network',
    'System',
    'Users',
    'Volumes',
    'bin',
    'cores',
    'dev',
    'home',
    'net',
    'opt',
    'private',
    'sbin',
    'usr' ]

```

It also gets you the hidden directories.
